## 導入方法

    $ git clone https://github.com/ferretdayo/question-problem.git
    
    $ cd question-problem
    
    $ composer install

    $ php artisan serve

    $ http://localhost:8000のはず

あとMySQLを利用しており，info_bbs.sqlに構造が入っているので，importしてください．
